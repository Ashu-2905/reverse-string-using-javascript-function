Reverse String Using JavaScript
A simple web-based JavaScript project that allows users to enter a string in the browser and instantly see the reversed string on the screen.
This project is beginner-friendly and demonstrates the use of functions, loops, and DOM manipulation.

 Features
Take string input from the user
Reverse the string using a JavaScript function
Display the result directly in the browser
Clean and responsive UI
Separate HTML, CSS, and JavaScript files
Easy to understand logic
